# Práctica Laravel - Gestión de Alumnos y Proyectos

## [cite_start]Descripción General [cite: 58]
Esta aplicación web permite la gestión administrativa de alumnos mediante un CRUD completo y la visualización de proyectos académicos. Cuenta con sistema de autenticación, diseño responsivo y traducción a tres idiomas (Español, Inglés, Francés).

* PHP >= 8.1
* Composer
* Node.js & NPM
* Base de datos MySQL/MariaDB
